# StudyDeck Forum

A discussion platform where students can create categories, ask questions, and share knowledge. Built with Django and deployed on PythonAnywhere.

## Live Demo
**View the live site here:** [https://tejalgoyal.pythonanywhere.com]

## Features
* **Create Categories:** Users can organize discussions by subject (e.g., Math, Physics).
* **Start Discussions:** Users can post questions inside categories.
* **Database Management:** Uses SQLite for local development and PythonAnywhere for production.

## Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML, CSS (Django Templates)
* **Database:** SQLite3
* **Deployment:** PythonAnywhere
